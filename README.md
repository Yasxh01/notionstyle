# Notion-Style Productivity Editor

A modern, feature-rich block-based editor inspired by Notion, built with React 19, Context API, and Tailwind CSS.

## ✨ Features

- Block-based editing (text, headings, lists)
- Drag-and-drop reordering
- Real-time updates with localStorage persistence
- Keyboard shortcuts (Ctrl/Cmd + S, Enter, Backspace)
- Export as Markdown or plain text
- Responsive design
- Dark mode support
- Optimized performance

## 🚀 Quick Start

```bash
pnpm install
pnpm dev
